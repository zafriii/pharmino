# Sale Feature

This folder contains all sale-related functionality including:

## Components
- `TabletSellSection.tsx` - Custom tablet selling interface
- `SalePanelEnhanced.tsx` - Enhanced sale panel with tablet support

## Utils
- `tablet-calculation.utils.ts` - Tablet price and inventory calculations
- `inventory-tablet.utils.ts` - Tablet-level inventory management

## Types
- `tablet-sale.types.ts` - Types for tablet selling functionality

## Features
- Single tablet selling for products with `tabletsPerStrip`
- Automatic price calculation based on `pricePerUnit`
- Tablet-level inventory deduction
- Custom checkbox UI for tablet selection
